# Beaked-Whale-Abundance-Estimation
Estimate beaked whale density and abundance from acoustic surveys using drifting recorder systems.  This study incorporates code from Taiki Sakai's [RoboJ package](https://github.com/taikisan21/roboj), building on methods described in Barlow et al 2021 (https://doi.org/10.1111/mms.12872)
